# Arrow Function

## Function Declaration

```javascript
function tampilPesan(name) {
  alert('Hello ' + name );
}

tampilPesan('Yusril');
```

## Function Expression

```javascript
let tampilPesan = function(nama) {
  alert('Hello ' + nama);
}

tampilPesan('Yusril');
```

## Definisi

> Bentuk lain yang lebih ringkas dari Function Expression
>
> - __MDN_

## Arrow Function

```javascript
let tampilPesan = (nama) => {
  alert('Hello ' + nama);
}

tampilPesan('Yusril');
```
